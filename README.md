# Project Template

Template com Claude GitHub Action pré-configurado.

## Funcionalidades

- Claude Code integrado via GitHub Actions
- Responde a `@claude` em Issues e PRs

## Como Usar

1. Crie um novo repo usando este template
2. Edite o `CLAUDE.md` com as instruções do seu projeto
3. Mencione `@claude` em qualquer Issue ou PR

## Exemplos de Uso

```
@claude implemente a função de autenticação descrita nesta issue

@claude revise este PR procurando bugs

@claude como devo estruturar este componente?
```

## Requisitos

- Secret `ANTHROPIC_API_KEY` configurado na organização
- Claude GitHub App instalado
